## FAQ

**1) How do I know the VPN is actually on?**  
Connect, then check your IP/location in the app and confirm your public IP changed in a browser. If it didn’t change, you’re not tunneled.

**2) Should I leave Kill Switch enabled?**  
Yes. If the connection drops, the Kill Switch prevents leaks by blocking traffic outside the tunnel.

**3) Which server should I pick for speed?**  
Start with the closest region. If it’s congested, switch to a nearby city or “recommended/fastest” server.

**4) Will this slow down my internet?**  
Some overhead is normal, but a nearby server with a stable connection should stay fast. Big slowdowns usually mean distance, congestion, or local network issues.
